## ✨ Overview  

A clean and elegant data preprocessing project focused on transforming raw shark attack data into a structured dataset ready for analysis and machine learning.

---

## 📊 What This Project Does  

✔️ Cleans messy real-world data  
✔️ Handles missing & inconsistent values  
✔️ Extracts useful features (like age)  
✔️ Standardizes categorical data  
✔️ Prepares dataset for analysis & ML  

---

## 🧹 Cleaning Pipeline  

### 🔹 Column Cleanup  
→ Removed whitespace & standardized column names  

### 🔹 Missing Values  
→ Filled missing entries with `"Unknown"`  

### 🔹 Duplicates Removed  
→ Ensured data consistency  

### 🔹 Feature Engineering  
→ Extracted numeric age using regex  
→ Cleaned and formatted date fields  

### 🔹 Categorization  
→ Grouped activities (Swimming, Surfing, etc.)  

### 🔹 Location Cleaning  
→ Standardized messy location strings
